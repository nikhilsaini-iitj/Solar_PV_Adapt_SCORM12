Client Need This Module Solves

Field technicians and sales teams need to quickly read a solar panel datasheet, understand the difference between mono / poly / thin-film, and confidently explain module construction to customers. This module replaces a static PDF deck with a short, trackable, device-friendly learning unit that works inside any SCORM 1.2 LMS.

Scene Breakdown

Video 1. Welcome — Choosing the Right Solar Panel ~1:40

Learning Objective

  • Hook the learner with a real rooftop scenario.
  • State what the module will cover.

Visual Treatment

  • Cinematic dusk rooftop with silhouette figures.
  • Animated "Simulated output" energy widget.
  • Gold corner brackets and deep navy palette.
Video Interactive 2. From a Cell to a Module ~2:10

Learning Objective

  • Explain how individual cells are wired and laminated into a module.
  • Connect cell behaviour to real-world module output.

Interactive Element

  • Embedded P-N junction simulation: fire a photon and watch electron-hole separation.
  • Launch button opens full-screen simulation.
Video Interactive 3. Inside the Module ~2:00

Learning Objective

  • Name the layers of a PV module from glass to frame.
  • Understand why each layer matters for 25-year outdoor life.

Interactive Element

  • 3D virtual lab: rotate, explode layers, switch cell technology.
  • System energy flow diagram (day/night switching).
Video Interactive 4. The Three Technologies ~1:50

Learning Objective

  • Compare monocrystalline, polycrystalline, and thin-film.
  • Match each technology to its best site condition.

Interactive Element

  • Narrative swipe cards with visual comparison.
  • Panel type selector inside 3D lab.
Video Interactive 5. Reading Specifications ~1:40

Learning Objective

  • Decode Pmax, Voc, Isc, Vmp, Imp, efficiency, and temperature coefficient.
  • Link datasheet numbers to on-site performance.

Interactive Element

  • Panel parts + specification card with clickable hotspots.
  • Full-screen launch from the embedded interactive.
AR 6. AR Experience — Place the Panel in Your Space ~2:00

Learning Objective

  • Let the learner visualise a life-size panel on a rooftop or floor.
  • Reinforce physical scale and placement constraints.

How It Works

  • iPhone Safari → AR Quick Look via panel.usdz.
  • Android Chrome → model-viewer WebXR / Scene Viewer.
  • Desktop → 3D preview + QR code to open on phone.
Assessment 7. Knowledge Check ~1:00

Questions

  • Q1 MCQ: Which layer faces the sun?
  • Q2 MCQ: Which technology produces most power per m²?
  • Q3 Matching: Pmax / Voc / Isc meanings.

SCORM Reporting

  • Score raw is sent to the LMS.
  • Lesson status flips to completed on pass.
  • Interactions are recorded per question.
